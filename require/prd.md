# **🏁 Hillyheally – 제품 기획서 (PRD) v1.4**

***“HILL YOUR WAY, HEAL YOUR WAY”***

---

## **1. 제품 개요**

Hillyheally는 러닝, 트레일러닝, 사이클 등 **아웃도어 대회 플랫폼**으로,

**대회 신청 → 참가 → 실시간 관제 → 기록 관리**까지 전 과정을 지원합니다.

또한, **한국 주요 트레일 정보를 3D 지도**로 제공하며,

**하이킹 커뮤니티/챌린지 기능**을 통해 유저의 지속적 성장과 몰입을 돕습니다.

**🚀 핵심 차별화 요소**

- **트래픽 병목 없는 대회 신청 시스템**
    - 가상 마라톤 대회
    - 가상 올림픽 대회
- **3D 지도 기반 실시간 GPS 트래킹**
- **트레일 콘텐츠 판매 + 커뮤니티 기능**

## **3. 문제 정의**

- 신청/결제 시스템 분산 → 사용자 혼란
- 트래픽 집중 → 서버 지연 및 결제 실패
- 참가자 GPS 인증 및 기록 기능 부족
- 주최자 관제/정산/운영 시스템 미비
- 트레일 정보 파편화

## **6. 시스템 아키텍처 요약**

- **앱**: Swift / Kotlin + Vue 웹뷰
- **서버**: JAVA REST API
- **DB**: MySQL or PostgreSQL
- **결제**: 토스페이먼츠 연동
- **푸시 알림**: FCM / APNs
- **트래픽 대응**: Redis 기반 큐잉 + 이중 등록 방지

---

## **7. 유저 플로우 예시**

## **12. 벤치마크 분석 – Follow My Challenge (FMC)**

### **📌 회사 정보**

| **항목** | **내용** |
| --- | --- |
| 설립 | 2015년, 네덜란드 |
| 본사 | 프랑스 Montignac-Lascaux |
| 대상 | 사이클, 하이킹, 요트레이스, 마라톤 |
| 지역 | 유럽/미국/동남아 등 글로벌 |

### **🔧 핵심 제품**

- **실시간 GPS 트래킹 플랫폼**
    - 고도 차트, 리더보드, 날씨, SNS 연동
    - 코스 리플레이 / 대회 지도 공유
- **트래커 장비 제공**
    - GSM 기반 (GL320MG), 위성 기반 (SPOT Gen4)
    - 원격 제어, 위치 정확도 2.5m
- **개인 도전도 지원**
    - 개인 기네스 기록 도전, 비공식 챌린지 가능

### **💼 비즈니스 모델**

| **수익 구조** | **설명** |
| --- | --- |
| 트래커 대여 | 1~14일 기간제 |
| 지도 커스터마이징 | 스폰서, SNS, POI 삽입 |
| 온라인 등록/예약 | 참가자 직접 등록 |
| B2B 이벤트 트래킹 | 대회용 지도/트래커 패키지 |

### **🌍 차별점**

- 고정밀 트래커 + 독자 플랫폼 운영
- 지역 파트너(UK, USA 등)를 통한 글로벌 확장
- Garmin, SPOT 등 외부 장비와 연동 가능
- 개인 + 기업 대상 모두 지원 가능
