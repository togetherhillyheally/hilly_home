# 🏁 Hillyheally – 제품 기획서 (PRD) v1.5 (대회 + 지도)

**_“HILL YOUR WAY, HEAL YOUR WAY”_**

---

## 1. 제품 개요

Hilly Heally는 러닝, 트레일러닝, 하이킹, 사이클 등 **아웃도어 대회 및 지도 기반 플랫폼**입니다.

**대회 신청 → 참가 → 실시간 관제/기록 관리**를 지원하는 동시에,  
**3D 지도 기반 실시간 그룹 트래킹과 미션 기능**을 제공하여  
혼자 걷더라도 “함께 걷는 경험”을 만듭니다.

**🚀 핵심 차별화 요소**

- **트래픽 병목 없는 대회 신청·결제**
- **QR 참가증, 주최자 관제 시스템**
- **실시간 그룹 트래킹 (함께 걷기)**
- **3D 지도 기반 미션 (함께 찾기, 함께 그리기)**
- **체크인·캠프·후기 기반 커뮤니티 확장**

---

## 2. 제품 미션

> “지도 위에서 연결되고, 길 위에서 함께 성장한다.”

## 3. 문제 정의

- 대회 신청/결제 시 서버 지연·혼란

- 트레일/하이킹 정보 파편화
- 혼자 하는 아웃도어 활동의 몰입도 저하

## 7. 시스템 아키텍처 요약

- **앱**: React Native (iOS/Android)
- **서버**: JAVA REST API
- **DB**: PostgreSQL
- **지도**: 3D GPS 기반 지도
- **실시간**: Redis + WebSocket/SSE
- **결제**: 토스페이먼츠 (대회·지도 패키지)
- **알림**: FCM / APNs

---

## 8. 유저 플로우 예시

1. 대회/루트 탐색
2. 신청 및 결제 → QR 참가증 발급
3. 대회 참여 or 함께 걷기 모드 실행
4. 실시간 위치 확인 + 체크인/캠프 기록
5. 완주 후 배지 획득 + 후기 공유
